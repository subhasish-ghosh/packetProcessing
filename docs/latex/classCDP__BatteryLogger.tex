\doxysubsection{Detailed Description}
The Battery Monitor Logging Class. 

The class can be implemented to log into an xml file if required. Currently, it implements as stdout logging. The member functions can be overridden to class specific requirements. The logging can be compiled in debug mode (\+\_\+\+D\+E\+B\+UG) to retrieve debug logging.
